# TorIRC Perl Edition
il s'agit d'une simple réécriture de la version [python](https://github.com/GHOSTnew/TorIRC)

## Dépendances

* Perl >= 5.8.3
* IO::Socket::Socks
* IO::Socket

## Utilisation

lancez le :
```
$ perl TorIRC.pl
```

Puis connectez vous sur 127.0.0.1,  port 20000

Exemple avec [irssi](http://irssi.org/):
```
$ irssi -c 127.0.0.1 -p 20000
```
